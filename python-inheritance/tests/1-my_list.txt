====================
Tests for MyList module
====================

Import:
    >>> MyList = __import__('1-my_list').MyList

Normal case:
    >>> my_list = MyList()
    >>> my_list.append(1)
    >>> my_list.append(4)
    >>> my_list.append(2)
    >>> my_list.append(3)
    >>> my_list.append(5)
    >>> print(my_list)
    [1, 4, 2, 3, 5]
    >>> my_list.print_sorted()
    [1, 2, 3, 4, 5]
    >>> print(my_list)
    [1, 4, 2, 3, 5]

Empty list:
    >>> empty_list = MyList()
    >>> empty_list.print_sorted()
    []

Already sorted:
    >>> sorted_list = MyList()
    >>> sorted_list.append(1)
    >>> sorted_list.append(2)
    >>> sorted_list.append(3)
    >>> sorted_list.print_sorted()
    [1, 2, 3]

Negative numbers:
    >>> neg_list = MyList()
    >>> neg_list.append(-1)
    >>> neg_list.append(-5)
    >>> neg_list.append(3)
    >>> neg_list.print_sorted()
    [-5, -1, 3]

Single element:
    >>> single_list = MyList()
    >>> single_list.append(7)
    >>> single_list.print_sorted()
    [7]
